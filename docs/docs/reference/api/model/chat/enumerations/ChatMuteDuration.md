# Enumeration: ChatMuteDuration

Valid durations for muting a chat using [[muteChat]]

## Enumeration Members

### EIGHT\_HOURS

> **EIGHT\_HOURS**: `"EIGHT_HOURS"`

Mutes chat for 8 hours

***

### FOREVER

> **FOREVER**: `"FOREVER"`

Mutes chat forever

***

### ONE\_WEEK

> **ONE\_WEEK**: `"ONE_WEEK"`

Mutes chat for 1 week
